SplitsProfile.load_from_file(path: str)¶

Read content of splits file and load them into splits profile object. Each valid (non-comment) line in the file is turned into an element of the splits list in the generated splits profile object.

If the path is invalid, the splits profile turns out empty.

Parameters

path – (str) file path to splits file

Returns

(SplitsProfile) object representation of read in splits file
